## 项目路线图 [Roadmap](./Roadmap.md)

假设你已经有了 C#基础、ASP .NET MVC 或其他语言的 MVC 基础。

本项目以 C#语言为示例，结合 ASP .NET Core，集成第三方类库的示例，运用基础组件，写好 Demo。如果你是一个 .NET Framework 开发者，转去学习 .NET Core,你会发现新的世界，大家定位是应该软件开发工程师，而不只是 .NET 开发工程师。
